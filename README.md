ghost_export
==========

* ghost_export 是为Ghost 导出博客文件的一个python小工具,默认把ghost_export.py放在Ghost的根目录下,它就可以顺利工作了. 
* 导出 的目录为 post
* created by Wasabi 2013-10-18
* Version 0.1

<pre>
# 进入Ghost的目录
> cd Ghost

# 导出操作
> python ghost_export.py
</pre>


### sever_start 
* 直接打开npm start 的小工具. 放在Ghost的根目录下,直接运行就可以.
